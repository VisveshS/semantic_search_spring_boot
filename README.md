Simple _spring boot mvc_ in java for providing interface for local semantic search.

Semantic search code in python , and is used for searching best matches for query string from many phrases

IDE: IntelliJ

python backend scripts dealing with sqlite database and huggingface embedding model is documented here [github code for python backend](https://github.com/VisveshS/Semantic-Search-Python-e5base-)
package used: [txtai](https://neuml.github.io/txtai/examples/)
